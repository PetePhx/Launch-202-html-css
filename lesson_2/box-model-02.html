2. Given the code below, what is the minimum width and height (in pixels) that the div needs to entirely contain the section element (including its margins)? How does this differ from the result of the previous practice problem?

`div` and `section` have visual formatting models of border-box and content-box, respectively.

Unlike the previous exercise, the inner element here (`section`) has a block display property. This means that it will occupy all the available width of the parent element.

counting the height and width values for the `section`:

section height: 300 + 10 + 10 + 4 + 4 + 20 + 10 = 358

section width: 500 + 20 + 20 + 4 + 4 + 19 + 11 = 578

To these div borders and paddings are added to arrive at the div box size:

div height: 360px

div width: 580px

Unlike the previous exercise, the block element (section) will be the only element on the line by itself.

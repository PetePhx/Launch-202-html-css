3. Given the code below, what is the minimum width and height (in pixels) that the div needs to entirely contain the em element (including its margins)?

`div` and `em` have visual formatting models of border-box and content-box, respectively.

Unlike the previous exercise, the inner element here (`em`) has the inline display property. For inline elements, browsers ignore the width/height values (instead calculating based on the content) as well as the top/bottom margin values. The use of these properties for inline elements is not recommended though.

The content in this case being default font size, this gives us the starting point of 16px for default font size. For simplicity sake, we start by 50px height and 100px width, then calculating the height and width values for `em`:

em height: 50 + 10 + 10 + 4 + 4 = 78px

em width: 100 + 20 + 20 + 4 + 4 + 19 + 11 = 178px

To these div borders and paddings are added to arrive at the div box size:

div height: 80px

div width: 180px

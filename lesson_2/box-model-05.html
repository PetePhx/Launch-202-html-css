5. Which of the following element pairs will display side-by-side in the `div`? Select all that apply:

Both elements are block elements.
Both elements are inline elements. (yes)
Both elements are inline-block elements. (yes)
One element is a block element, and one is an inline element.
One element is a block element, and one is an inline-block element.
One element is an inline element, and one is an inline-block element. (yes)

You may assume that any inline element has a content width of no more then 360 pixels. Remember, the width property doesn't affect inline elements so this "content width" is the actual width of the content area as determined by your browser.

Answer:

The cases where we have any combinations of inline and/or inline-block elements, as these would allow for two items side by side fitting inside the `div` content-box. Block elements have to be each on a line by itself.

6. Will the following code display the two article boxes side-by-side? If not, why not? How would you fix it so that it places the boxes side-by-side?

Answer:

No, since `article` has a block element visual formatting model. To allow for side-by-side display, the display property has to be set to inline or inline-block.

The box-sizing property also needs to be set to border-box in order for the border thicknes to be absorbed into the width property of 50%, hence `section` width accomodating two side by side article itmes.
